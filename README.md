# CSIT_360-Recommendation-System-Project
Team Members: Nicholas Barrale, Paramjit Kaur  
Dataset: movie-ratings.txt  
The dataset we are using for the recommendation system is from CiaoDVD, which is a dataset crawled from the DVD category of the dvd.ciao.co.uk product review website in December 2013. Specifically, we will use the dataset “movie-ratings.txt.” This dataset contains a list of ratings by different users. It includes six features, those being userId, movieId, movie-categoryId, reviewId, the movie rating from 1 to 5, and the date of the review.
